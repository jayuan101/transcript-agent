# Transcript Agent

AI-powered transcription and analysis — Whisper + Claude.

**Bring your own Anthropic API key.** Usage is billed to your account, not the host's.

## What it does

- Transcribes audio, video, and documents
- Identifies and labels multiple speakers
- Generates summary, key points, and action items
- Detects speech rate and accent indicators
- Supports 28+ languages

## Supported formats

| Type | Formats |
|------|---------|
| Audio | mp3, wav, m4a, flac, ogg, aac |
| Video | mp4, mov, avi, mkv, webm |
| Docs | pdf, docx, txt, md, srt, vtt |

## How to use

1. Paste your [Anthropic API key](https://console.anthropic.com) — it is never stored
2. Upload or drag a file
3. Click **Analyze File**
